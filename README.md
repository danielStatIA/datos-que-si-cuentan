## Sobre el proyecto

Este repositorio reúne el material del libro y funciona como espacio de desarrollo, documentación y publicación.  
La idea es construir una ruta de aprendizaje útil para quienes quieren entender los datos con fundamento, trabajar con casos reales y avanzar hacia modelos más sofisticados de análisis y predicción.

Más que una colección de ejemplos, este proyecto busca acompañar un proceso de aprendizaje progresivo:

- partir de bases estadísticas sólidas,
- trabajar con datos reales siempre que sea posible,
- aprender a analizar, visualizar y modelar,
- y conectar la teoría con la práctica de forma natural.

## Objetivo

Este proyecto busca servir como una guía para:

- comprender fundamentos de estadística aplicada,
- limpiar, preparar y explorar datos,
- construir análisis reproducibles,
- desarrollar modelos de machine learning,
- introducir conceptos de deep learning,
- y sentar bases para aplicaciones de inteligencia artificial.

## Enfoque

La prioridad de este libro es aprender haciendo.

Por eso, el material está pensado para:

- usar datos reales en contextos reales,
- explicar cada paso con claridad,
- evitar ejemplos artificiales cuando existan alternativas auténticas,
- avanzar de forma progresiva,
- y mantener una base estadística antes de pasar a modelos más complejos.

## Cómo usar este repositorio

1. Clona el repositorio.
2. Instala las dependencias necesarias en R y Python.
3. Abre el proyecto en tu entorno de trabajo.
4. Ejecuta los capítulos y ejemplos.
